# 🏢 Real Estate Project Valuation & DCF Model

> Project Finance • DCF Valuation • Financial Modeling • Investment Appraisal

---

## 📌 Project Overview

This project evaluates the financial feasibility of a residential rental apartment development project using a Discounted Cash Flow (DCF) approach.

The objective is to determine whether the project creates value for investors by forecasting revenues, operating expenses, financing costs, and future cash flows. The analysis incorporates both the **Total Investment Point of View (TIPV)** and **Equity Point of View (EPV)** frameworks commonly used in project finance and investment appraisal.

The financial model was developed in Microsoft Excel and includes revenue forecasting, operating cash flow analysis, debt financing structure, valuation metrics, and sensitivity analysis.

---

## 🎯 Business Objective

The project aims to answer the following questions:

* Is the investment financially feasible?
* Does the project generate sufficient returns for investors?
* Can the project meet its debt repayment obligations?
* How sensitive is project performance to changes in key assumptions?

---

## 🏗️ Project Assumptions

### Property Development

* Land area: **32,000 m²**
* Apartment mix:

  * 60 one-bedroom units
  * 93 two-bedroom units
  * 22 three-bedroom units

### Investment Cost

* Total investment cost: **550,773 million VND**
* Construction cost: **14.4 million VND/m²**
* Additional costs:

  * Interior furnishing
  * Design fees
  * Insurance
  * Marketing expenses
  * Administrative costs

### Financing Structure

* Combination of equity financing and bank debt
* Debt repayment schedule integrated into the financial model
* Project evaluated under both TIPV and EPV perspectives

## ⚙️ Methodology

### Step 1: Revenue Forecasting

Projected rental revenues were estimated based on:

* Rental prices
* Occupancy assumptions
* Apartment types
* Inflation-adjusted growth

Revenue metrics include:

* Potential Gross Income (PGI)
* Effective Gross Income (EGI)

### Step 3: Financial Performance Analysis

The model forecasts:

* EBITDA
* EBIT
* Tax expenses

---

### Step 4: Free Cash Flow Estimation

The project calculates:

### TIPV (Project Cash Flow)

```text
Free Cash Flow to Firm (FCFF)
```

### EPV (Equity Cash Flow)

```text
Free Cash Flow to Equity (FCFE)
```

Both approaches are used to evaluate project viability.

### Step 5: Discounted Cash Flow Valuation

The project applies:

* WACC (Weighted Average Cost of Capital)
* NPV (Net Present Value)
* IRR (Internal Rate of Return)

to determine project value creation.

---

### Step 6: Debt Capacity Analysis

Debt service capability is assessed through:

```text
ADSCR
(Average Debt Service Coverage Ratio)
```

to evaluate the project's ability to meet loan obligations.

## 📊 Key Results

### TIPV Perspective

| Metric | Result             |
| ------ | ------------------ |
| NPV    | 27,130 million VND |
| IRR    | 10.63%             |
| WACC   | 10.07%             |

### Interpretation

The project generates a positive NPV under the total investment perspective, indicating that the project creates value after accounting for the required rate of return.

---

### EPV Perspective

| Metric | Result             |
| ------ | ------------------ |
| NPV    | 41,666 million VND |
| IRR    | 13.82%             |

### Interpretation

The project generates positive value for equity investors, with an IRR exceeding the required return of 12%.

---

### Debt Service Analysis

| Metric        | Result |
| ------------- | ------ |
| Average ADSCR | 2.07x  |

### Interpretation

The project demonstrates strong debt repayment capacity, with cash flows comfortably covering debt obligations throughout the investment period.
